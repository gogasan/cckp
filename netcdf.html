NetCDF File Conventions : Directory Structure and File Name

 

netCDF directory:

../{collection}/{variable}/{model-run}/{file with data}.nc

 

Collections:

  • cmip6              : oldest collection, equivalent to cmip6-1x1 (dims 181x361)
  • cmip6-x1         : new 1x1-degree resolution (dims 180x360)
  • cmip6-x0.5      : 0.5x0.5 degree CRU historical and ocean data
  • cmip6-x0.25    : 0.25x0.25 degrees from ERA5, and downscaled (721x1440)

 

Variable:

All variables created by CMIP6-D5a_Indicators_TimeseriesComputation.ncl

 

Model-Run:

$ModelName-$ModelVariant-$CMIPScenario
 

netCDF filename:

{product-variable}_{collection}_{model-run}_{type}_{percentile}_{period}.nc

$type-$variable-$aggregation-$statistic _ collection _ model-run _ $type _ $percentile _ period
